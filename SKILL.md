# 技能1：询问GM（梦幻精灵）

对于用户问的任何游戏问题，都需要先询问游戏GM获取相关信息。具体步骤：
1. 拆解用户的问题并提取问题内所有游戏名词
2. 询问GM对应名词的含义（请求形式可参考`scripts/gm_request.sh`）。如果GM的回复涉及新的名词知识，可以再次追问GM获取更多信息
3. 如有需要，调用后续其他技能帮助完成复杂计算
4. 回答用户问题

# 技能2：价格换算

梦幻西游这个游戏涉及了几个重要货币单位：银两（`tael`）、点卡（`point`）、元（`yuan`）、三界功绩（`merit`）、仙玉（`jade`）和储备金（`reserve`），它们之间可以互相转换。如果遇到转换相关的计算，需要先参考`scripts/price_converter.py`中对应的函数，然后按需询问用户当前的比例（`rate`，单位：元/万银两）或者点卡价格（`price`, 银两/点），最后调用函数计算。


# 技能3：体力转换

梦幻西游中的体力（`stamina`）也可以转换成银两（`tael`）和储备金（`reserve`）。如果被用户询问体力的价值，可以参考`scripts/stamina_calculator.py`中对应的函数计算。

# 技能5：宝石

计算宝石价格(`price`)时，需要先询问用户该宝石的等级，以及其对应的1级宝石价格是多少。计算宝石合成所需体力（`stamina`）时，需要先询问用户该宝石的等级。然后调用`scripts/gem_calculator.py `中对应的函数计算宝石价格和体力消耗。宝石一共有三种类型：
1. 普通宝石（gem）：光芒石、太阳石、月亮石、红宝石、黄宝石、蓝宝石、绿宝石、黑宝石、红玛瑙、舍利子、神秘石、翡翠石
2. 星辉石（star stone）
3. 五色灵尘（spirit dust）
